DESARROLLO DE SOFTWARE

El desarrollo de software es un proceso, aplicado directamente al desarrollo de un producto se software, el cual, se lleva a cabo por medio de unos procesos específicamente estructurados, donde cada proceso tiene un bloque de actividades organizadas y planificadas, dichas actividades se plantean con el fin de tener un producto final, que cumpla las necesidades y requerimientos de un usuario final, es decir el cliente.
Esta ova tiene como propósito, dar a conocer dichas etapas, implementadas en un proceso de desarrollo de software, las cuales están descritas en el menú principal de esta página web.
